# Skill: Context Engineering & Memory Systems

Source: muratcankoylan-agent-skills-for-context-engineering-memory-systems (via SkillHub)

Advanced context management and memory system design.

## Commands

- Context design: Structure context for optimal model performance
- Memory management: Implement short and long-term memory
- Context compression: Reduce context size while preserving meaning
- Retrieval strategies: Optimize context retrieval

## Context Engineering Principles

1. **Relevance**
   - Include only relevant information
   - Filter out noise
   - Prioritize recent/important data

2. **Structure**
   - Organize context logically
   - Use clear delimiters
   - Maintain hierarchy

3. **Compression**
   - Summarize when possible
   - Remove redundancy
   - Use efficient representations

## Memory Systems

- **Short-term Memory**
  - Current conversation context
  - Active task information
  - Temporary variables

- **Long-term Memory**
  - Persistent knowledge
  - Learned patterns
  - User preferences

## Techniques

- Windowing: Sliding context windows
- Summarization: Compress long contexts
- Indexing: Fast retrieval structures
- Prioritization: Rank information importance
