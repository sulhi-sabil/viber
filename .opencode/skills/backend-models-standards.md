# Skill: Backend Models & Standards

Source: maxritter-claude-codepro-backend-models-standards (via SkillHub)

Backend development standards and model best practices.

## Commands

- Design APIs: Create REST/GraphQL API specifications
- Model design: Define data models and relationships
- Code standards: Follow backend coding conventions
- Architecture review: Evaluate backend architecture

## API Design Principles

- **RESTful Design**
  - Use proper HTTP methods
  - Resource-based URLs
  - Consistent naming conventions
  - Proper status codes

- **GraphQL Best Practices**
  - Schema-first design
  - Proper type definitions
  - Query optimization
  - Error handling

## Model Standards

- Clear entity definitions
- Proper relationship mapping
- Validation rules
- Migration strategies

## Code Standards

- Layered architecture (Controller/Service/Repository)
- Dependency injection
- Error handling patterns
- Logging standards
- Security practices

## Anti-Patterns to Avoid

- God objects
- Circular dependencies
- Business logic in controllers
- SQL injection vulnerabilities
- Hardcoded configuration
