# Skill: Debugging Strategies

Source: madappgang-claude-code-debugging-strategies (via SkillHub)

Systematic approaches to debugging complex issues.

## Commands

- Root cause analysis: Trace issues to their source
- Binary search debugging: Isolate problematic code sections
- Hypothesis testing: Form and test debugging hypotheses
- Defense in depth: Multiple layers of debugging checks

## Debugging Workflow

1. **Reproduction**
   - Create minimal reproducible example
   - Verify issue is consistent
   - Document exact error conditions

2. **Isolation**
   - Identify affected components
   - Narrow down code section
   - Check dependencies

3. **Hypothesis Formation**
   - Generate possible causes
   - Prioritize by likelihood
   - Plan validation tests

4. **Testing & Validation**
   - Test each hypothesis
   - Verify fixes work
   - Check for side effects

5. **Documentation**
   - Record root cause
   - Document solution
   - Update runbooks

## Techniques

- Print debugging (strategic logging)
- Rubber duck debugging
- Binary search bisection
- Unit test isolation
- Memory profiling
- Performance profiling
